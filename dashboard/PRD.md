# PRD: 주식 리서치 대시보드

## 1. 개요

### 제품 목적
관심종목 주가·재무 정보, 거시경제 변수, 주요 이벤트 일정을 한 화면에서 빠르게 파악할 수 있는 개인용 주식 리서치 대시보드.

### 대상 사용자
개인 투자자 (주식 리서치 목적)

### 기술 스택 (예정)
- Frontend: React + TailwindCSS
- 차트: Recharts 또는 Chart.js
- 데이터: FinanceDataReader, DART API, Yahoo Finance API
- 캘린더: FullCalendar.js

---

## 2. 화면 구조

상단 네비게이션 바에서 아래 3개 메뉴를 탭으로 선택:

```
[ 관심종목 ]  [ 거시변수 ]  [ 캘린더 ]
```

---

## 3. 메뉴별 요구사항

### 3-1. 관심종목

#### 목적
등록된 관심종목의 주가, 시가총액, 주요 재무지표를 한눈에 볼 수 있는 대시보드.

#### 기능 요구사항

| ID | 기능 | 설명 |
|----|------|------|
| W-01 | 종목 등록/삭제 | 티커 심볼로 관심종목 추가·제거 |
| W-02 | 주가 요약 카드 | 현재가, 등락률, 52주 고/저 표시 |
| W-03 | 시가총액 표시 | 종목별 시총 (단위: 조/억) |
| W-04 | 재무제표 요약 | 매출액, 영업이익, 순이익, PER, PBR, ROE (최근 4분기) |
| W-05 | 주가 차트 | 종목별 1년 주가 차트 (미니 스파크라인) |
| W-06 | 정렬/필터 | 시총, 등락률, PER 기준 정렬 |

#### 화면 레이아웃
```
┌─────────────────────────────────────────────┐
│  + 종목 추가      정렬: [시총 ▼]            │
├──────┬────────┬────────┬────────┬────────────┤
│ 종목 │ 현재가 │ 등락률 │  시총  │ PER / PBR  │
├──────┼────────┼────────┼────────┼────────────┤
│ AAPL │ $189  │ +1.2%  │ 2.9조$ │ 28x / 8.2x │
│ TSLA │ $245  │ -0.8%  │ 0.8조$ │ 65x / 12x  │
│  ... │  ...  │  ...   │  ...   │    ...     │
└──────┴────────┴────────┴────────┴────────────┘

[ 선택 종목 재무제표 상세 패널 ]
매출액 | 영업이익 | 순이익 | FCF  (최근 8분기 바차트)
```

#### 데이터 소스
- 주가: FinanceDataReader (`fdr.DataReader`)
- 재무제표: DART API / Yahoo Finance (`yfinance`)

---

### 3-2. 거시변수

#### 목적
주요 거시경제 지표와 시장 인덱스를 최근 5년 차트로 한눈에 확인.

#### 화면 레이아웃
3열 × 2행 = **6개 차트 패널** (각 패널: 지표명 + 5년 라인차트)

```
┌──────────────┬──────────────┬──────────────┐
│ 미국 10Y 금리 │   S&P 500   │  달러 인덱스  │
│  [차트 5Y]   │  [차트 5Y]  │  [차트 5Y]   │
├──────────────┼──────────────┼──────────────┤
│  WTI 유가    │  VIX 지수   │  나스닥 100  │
│  [차트 5Y]   │  [차트 5Y]  │  [차트 5Y]   │
└──────────────┴──────────────┴──────────────┘
```

#### 기본 6개 지표

| 패널 | 지표명 | FinanceDataReader 코드 |
|------|--------|----------------------|
| 1 | 미국 10년물 국채금리 | `fdr.DataReader('US10YT=X')` |
| 2 | S&P 500 | `fdr.DataReader('SP500')` |
| 3 | 달러 인덱스 (DXY) | `fdr.DataReader('DX-Y.NYB')` |
| 4 | WTI 원유 | `fdr.DataReader('CL=F')` |
| 5 | VIX (공포지수) | `fdr.DataReader('VIX')` |
| 6 | 나스닥 100 | `fdr.DataReader('QQQ')` |

#### 기능 요구사항

| ID | 기능 | 설명 |
|----|------|------|
| M-01 | 5년 라인차트 | 각 패널에 최근 5년 일별 데이터 표시 |
| M-02 | 기간 선택 | 1Y / 3Y / 5Y 버튼으로 기간 변경 |
| M-03 | 현재값 표시 | 각 패널 상단에 현재 수치 및 전일 대비 등락 표시 |
| M-04 | 지표 교체 | 6개 패널 각각 원하는 지표로 변경 가능 |
| M-05 | 데이터 갱신 | 페이지 로드 시 자동 갱신, 수동 새로고침 버튼 |

### 3-3. 캘린더

#### 목적
관심종목 실적 발표, Fed 회의, 주요 경제지표 발표 일정을 달력에서 한눈에 확인.

#### 화면 레이아웃
```
┌─────────────────────────┬──────────────────────┐
│                         │  이벤트 필터         │
│   월간 캘린더           │  ☑ Earnings Call     │
│                         │  ☑ Fed Meeting       │
│   (이벤트 아이콘 표시)  │  ☑ CPI 발표          │
│                         │  ☑ PPI 발표          │
│                         │  ☑ 고용보고서        │
│                         │  ☑ PCE 발표          │
│                         │  ☑ GDP 발표          │
│                         │  ☑ FOMC 의사록       │
│                         │  ☑ ISM 제조업PMI     │
│                         │  ☑ 소비자신뢰지수    │
└─────────────────────────┴──────────────────────┘
```

#### 이벤트 10종 (체크박스로 개별 ON/OFF)

| # | 이벤트 | 색상 태그 |
|---|--------|----------|
| 1 | Earnings Call (관심종목) | 🔵 파랑 |
| 2 | Fed Meeting (FOMC) | 🔴 빨강 |
| 3 | CPI 발표 | 🟠 주황 |
| 4 | PPI 발표 | 🟡 노랑 |
| 5 | 고용보고서 (Non-Farm Payroll) | 🟢 초록 |
| 6 | PCE 발표 | 🟣 보라 |
| 7 | GDP 발표 | 🟤 갈색 |
| 8 | FOMC 의사록 공개 | 🔴 빨강(연) |
| 9 | ISM 제조업 PMI | ⚫ 회색 |
| 10 | 소비자신뢰지수 (Conference Board) | 🩵 하늘 |

#### 기능 요구사항

| ID | 기능 | 설명 |
|----|------|------|
| C-01 | 월간 캘린더 뷰 | 기본 월간 보기, 주간 보기 전환 가능 |
| C-02 | 이벤트 필터 | 우측 체크박스로 이벤트 종류별 표시/숨김 |
| C-03 | 이벤트 클릭 | 클릭 시 상세 정보 팝업 (종목명, 예상 EPS 등) |
| C-04 | 관심종목 연동 | 관심종목 탭에 등록된 종목의 실적 일정 자동 반영 |
| C-05 | D-day 표시 | 다가오는 이벤트까지 남은 일수 표시 |

#### 데이터 소스
- Earnings Call: Yahoo Finance earnings calendar API
- Fed/경제지표: Federal Reserve 공식 캘린더, Investing.com
